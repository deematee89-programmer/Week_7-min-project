W7 min project Analysis Pipeline
========================================================================================
This project implements a structured exploratory data analysis process to
understand the dataset and identify key patterns and relationships.
----------------------------------------------------------------------------------------
Steps Performed:
1-Loaded the dataset and inspected its structure
2-Performed univariate analysis using histograms for numeric features (age, income, spend)
3-Conducted bivariate analysis using boxplots across the segment category
4-Computed correlation matrix for numeric features
5-Visualized correlations using a heatmap and scatter plots (age vs income)
-----------------------------------------------------------------------------------------
Insights:
1-Income and spend have high values that may act as outliers
2-The segment category affects the distribution of numeric values
3-Income and spend are positively correlated; older or wealthier customers tend to spend more
